# AGENTS.md

Guidance for agents working in this repository.

## Core Ethos

Otto exists to reduce phone usage time. The home screen and adjacent flows should be boring, low-stimulation, and deliberately frictional to use. Avoid colorful, entertaining, promotional, or convenience-driven UI unless it clearly supports leaving the phone faster.

## Working Notes

- Keep launcher surfaces minimal and quiet.
- Prefer hidden gestures or small controls over large attention-grabbing actions.
- Preserve useful friction around opening distracting apps and settings.
- Do not add telemetry, gamification, streaks, rewards, or visual flourishes.
